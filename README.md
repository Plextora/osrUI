# Features
- Edit hit judgements (number of 300s, number of 50s, etc.)
- Edit mods in replay
- Edit replay username
- Edit score
- And much more!

# How do I install this?
Head over to the [latest release page](https://github.com/Plextora/osrUI/releases/latest) to get the setup exe.
